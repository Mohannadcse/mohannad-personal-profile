My research focuses on program analysis and cybersecurity, utilizing static and dynamic analysis, symbolic execution, and formal verification to assess application safety and security in emerging domains.

## Research Areas

### IoT Safety and Privacy
Developing automated techniques to detect security vulnerabilities in IoT systems, focusing on inter-app communication and interaction threats.

- **Selected Papers:** [USENIX'22](https://www.usenix.org/system/files/sec22-chen-yunang-practical.pdf), [IEEE TSE'22](https://ieeexplore.ieee.org/document/9785922), [ISSTA'20](https://ieeexplore.ieee.org/document/9785922) 🏆

### Software Debloating
Removing unnecessary code from applications to reduce attack surface and improve performance through compiler-assisted specialization.

- **Selected Papers:** [SIGMETRICS'24](https://arxiv.org/abs/2212.09437), [EuroS&P'22](https://ieeexplore.ieee.org/abstract/document/9797349), [FEAST'24]
- **Tools:** [FitStack](https://www.fitstack.dev) (commercialized), LMCAS, SLASH (ONR Tech Transfer)

### Machine Learning Robustness
Investigating adversarial attacks and defenses for ML systems, analyzing attack surfaces in bloated ML dependencies.

- **Selected Papers:** [NeurIPS'22](https://openreview.net/pdf?id=WBp4dli3No6), [SIGMETRICS'24](https://arxiv.org/abs/2212.09437)

### Android Security
Analyzing security vulnerabilities in Android applications, particularly inter-app communication and dynamically loaded code.

- **Selected Papers:** [INFOCOM'19](https://cse.unl.edu/~hbagheri/publications/2019INFOCOM.pdf), [IEEE TIFS'20]
